Set up a New Project

To create a new project, you will need to go to the Projects page. To access the Projects page, click on the Projects option from the navigation bar on the left-hand side.

You will see a similar screen.

Click on the Create New Project button in the top right corner to initiate a new project. A similar screen will appear.

Fill in all the required information, i.e.:

  1. Project Name: The name you want to give to your new project
  2. Project Description: Describe your project and what it aims to accomplish.
  3. Project Settings:
    • Upload Sources: Select the sources that your custodians can upload files from, such as selecting Local File will allow the custodian to upload files directly from their system, and Google Drive will allow the custodian to upload files via their Google Drive.
    • Target Source: Select the storage location you wish the files to be uploaded to.
    • AWS Bucket/Azure Container: Select or add information about your target source.
    • Enable File Verification: Select to enable verification of the files uploaded by custodians.

After you have added all the information, the Create Project button will become active. Click on the button to create the project.
If you wish to discard the project, click on the Cancel button to cancel creating the project. No information will be saved in this case.
After you have successfully created your project, it will appear in your Projects view with a status of Not Started indicating that it currently does not have any custodians and hence no activity performed on it.

For each project:

  1. Click on the View button to view the details of the project.
  2. Click on the gear icon to edit the project details.
