Configure AWS Buckets and Azure Containers

You can select your target source to be either AWS or Azure.

AWS Buckets

For AWS, you need to select a bucket to be the source. Select AWS and click on the pencil icon to edit and select a bucket other than the default one:

You will be prompted to either select an existing bucket or create a new one:

If you select Choose existing bucket, you will see a list of available options that you can choose from. Select any of your choice and click on the Save button.

You will now be able to see the selected bucket name in the target source.

However, if you choose to create a new bucket, select that option and click on the Save button.

A new bucket will be automatically created and selected for you.

Azure Containers

If you choose Azure as your target source, you will need to select the container in which the files of the project will be uploaded.
Select Azure and click on the pencil icon to edit and select a container other than the default one:

You will be prompted to select a container from the list of available containers. Select the one of your choice and click on the Save button:

You will now be able to see the selected container name in the target source.
